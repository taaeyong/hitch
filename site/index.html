Hitch

사랑은 있는데 자꾸 어긋나는 두 사람에게, 서로에게 더 잘 닿는 표현 방식을 찾아줍니다.

어떻게 돕나요 작동 방식
로컬 실행

복사해서 바로 시작

git clone https://github.com/taaeyong/hitch.git
cd hitch
uv sync
cp .env.example .env
# .env에서 HITCH_TELEGRAM_TOKEN만 교체
uv run hitch bot

소개

Hitch가 돕는 순간

서로를 사랑하지 않는 것이 아니라, 사랑을 느끼는 방식이 달라서 어긋나는 순간이 있습니다. Hitch는 그 차이를 부드럽게 드러내고, 이번 주에 해볼 수 있는 작은 행동으로 바꿔줍니다.

짧은 질문

부담 없는 선택형 질문으로 사랑을 느끼는 방식을 살펴봅니다.

감정 회고

최근 사랑받았다고 느낀 순간을 한두 문장으로 남깁니다.

차이 발견

두 사람의 표현 방식과 기대가 어디서 달라지는지 정리합니다.

작은 제안

이번 주에 바로 시도할 수 있는 관계 코칭을 건넵니다.

Hackathon systems diagram

Harness as an orchestrated AI system

The harness separates goal definition, implementation, and alignment into distinct roles. The result is not one model doing everything; it is a layered system coordinating specialized agents toward a shippable target.

Alignment monitor Blocker resolver Process steward
Atlas / OpenClaw Monitor the process

Resolve blockers, check progress against the contract, and keep the system aligned.

Oboros Define the target

Interview the user, extract the seed, and produce the acceptance contract.

Acceptance contract Shared target state
  • Intent captured through interview
  • Seed translated into scope
  • Done criteria made explicit
Codex Implement the target

Turn the acceptance contract into working code, tests, and deliverable artifacts.

Role separation Orchestration over raw generation Reliable shipping loop

작동 방식

대화는 가볍게, 정리는 조용하게

  1. 시작하기 텔레그램에서 Hitch를 시작합니다.
  2. 질문 답하기 짧은 질문에 답하며 나의 사랑 언어를 확인합니다.
  3. 순간 남기기 최근 사랑받았다고 느낀 순간을 기록합니다.
  4. 제안 받기 Hitch가 두 사람의 차이를 정리하고 작은 실천을 제안합니다.
